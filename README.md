Grama-Sanjeevini 🏥💊

Smart Medical Finder Android Application built using Kotlin, Jetpack Compose, and Firebase.
This application helps users search and locate medicines available in nearby pharmacies with stock availability and location details.

🚀 Features
    🔐 Firebase Login Authentication
    💊 Medicine Search Functionality
    🏪 Pharmacy Name Display
    📍 Bangalore Area-Based Medicine Availability
    🎨 Modern Medical UI Design
    🔎 Dynamic Search Bar
    📱 Jetpack Compose UI
    ☁️ Firebase Firestore Integration
    🧾 Large Medicine Database (100+ medicines)

🛠️ Technologies Used
      Kotlin
      Jetpack Compose
      Android Studio
      Firebase Authentication
      Firebase Firestore


📱 Screens Included

▶️ How to Run
Clone the repository
git clone https://github.com/Spandanamb7/gramasanjeevini.git
Open project in Android Studio
Connect Firebase
Add google-services.json inside:app/
Sync Gradle
Run the app on Emulator or Android Device
